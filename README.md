# Tarea Módulo 2: Desarrollo de Interfaces

## Descripción

Este proyecto consiste en una interfaz gráfica de gestión de usuarios desarrollada como parte del Módulo 2 de la asignatura de Desarrollo de Interfaces. El objetivo principal es poner en práctica el diseño de ventanas y componentes adaptativos, empleando buenas prácticas de usabilidad y diseño responsive.

## Objetivos

- Desarrollar una aplicación gráfica de escritorio para la gestión de usuarios.
- Demostrar la capacidad de adaptación de la interfaz a diferentes tamaños de ventana (responsive).
- Utilizar componentes de diálogo (JDialog) para tareas secundarias y modales.
- Mejorar la experiencia de usuario con un diseño claro y funcional.

## Tecnologías empleadas

- **Lenguaje:** Java
- **Framework:** Swing para la interfaz gráfica
- **IDE recomendado:** IntelliJ IDEA / NetBeans / Eclipse

## Cómo ejecutar el proyecto

1. Clona este repositorio:
   ```bash
   git clone https://github.com/AdanGavira/TareaModulo2_DesarrolloInterfaces.git
   ```
2. Abre el proyecto en tu IDE Java favorito.
3. Asegúrate de tener instalado JDK 8 o superior.
4. Ejecuta la clase principal (`Main` o similar) para lanzar la aplicación.
